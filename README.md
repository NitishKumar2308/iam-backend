# IAM Backend

This is a secure Identity and Access Management (IAM) backend built using Node.js and Express.

## Features
- Helmet, CORS, and Rate Limiting for security
- Environment configuration via .env
- Ready for Database and Authentication integration

## Run (when imported into Replit or Vercel)
1. Set environment variables.
2. Click “Run” in Replit or deploy via Vercel.

✅ Ready to integrate database and authentication next.
